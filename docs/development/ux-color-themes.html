Dark Mode Strategy

Implementation

  • Tailwind's dark: prefix for all utilities
  • Toggle via <html class="dark">
  • localStorage persistence
  • Respects system preference (prefers-color-scheme)

Dark Mode Adjustments

  • Lightened primary colors for contrast
  • Elevated surfaces (cards lighter than background)
  • Reduced shadows, adjusted for visibility
  • All text meets WCAG AA on dark backgrounds

Color Contrast Compliance

Light Mode
  • Primary on White: 15.7:1 (AAA)
  • Foreground on White: 12.6:1 (AAA)
  • Muted text: 5.7:1 (AA)
Dark Mode
  • Foreground on Dark BG: 14.2:1 (AAA)
  • Secondary on Dark BG: 7.8:1 (AAA)
  • Muted text: 4.6:1 (AA)
